Today's key insights:

1. **Two Pointer Pattern - Left/Right Convergence**: Valid Palindrome uses two pointers starting from both ends, moving toward center. O(n) time, O(1) space. Key insight: Skip non-alphanumeric characters while comparing.

2. **Two Pointer Pattern - Read/Write Position**: Move Zeroes uses one pointer to track "zero position" and another to iterate. When non-zero found, swap with zero position. O(n) time, O(1) space. Maintains relative order of non-zero elements.
